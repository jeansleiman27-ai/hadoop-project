\# Hadoop Big Data Project



\## Overview

This project demonstrates a complete Big Data pipeline using Hadoop ecosystem tools.



\## Used

\- Hadoop HDFS

\- Sqoop

\- Hive

\- Pig

\- Oozie

\- Docker



\## Data Pipeline Flow

1\. MySQL → Sqoop → HDFS

2\. Hive SQL analysis

3\. Pig data transformation

4\. Oozie workflow orchestration



\## Dataset

Simple student dataset:

\- id

\- name

\- grade
